## 1. 로그인 화면

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                    [ 로고 / 서비스명 ]                    │
│                  직원 업무 관리 시스템                     │
│                                                         │
│              ┌─────────────────────────────┐             │
│              │ 아이디 (이메일 또는 사번)     │             │
│              └─────────────────────────────┘             │
│              ┌─────────────────────────────┐             │
│              │ 비밀번호                     │             │
│              └─────────────────────────────┘             │
│                                                         │
│              [        로그인        ]                    │
│                                                         │
│              비밀번호를 잊으셨나요?                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**요소**
- 아이디 입력 (placeholder: 이메일 또는 사번)
- 비밀번호 입력 (마스킹)
- 로그인 버튼
- 비밀번호 찾기 링크 (선택)

## 6. 일정 상세 / 등록 (모달)

```
┌─────────────────────────────────────────────────────────┐
│  새 일정 등록                                    [X]     │
├─────────────────────────────────────────────────────────┤
│  제목 *     ┌─────────────────────────────────────────┐  │
│             │ 예: 주간 마케팅 회의                     │  │
│             └─────────────────────────────────────────┘  │
│  설명       ┌─────────────────────────────────────────┐  │
│             │                                         │  │
│             │ (선택)                                  │  │
│             └─────────────────────────────────────────┘  │
│  담당자 *   [본인 ▼]  (대표: [직원 선택 ▼])              │
│  팀         [마케팅팀 ▼]                                │
│  시작       [2025-02-23] [09:00]                        │
│  종료       [2025-02-23] [10:00]                        │
│  반복       [없음 ▼] (없음/매일/매주/매월)               │
│  우선순위   ( ) 낮음  (•) 보통  ( ) 높음                 │
│  상태       [예정 ▼] (예정/진행중/완료/보류)             │
│  알림       [10분 전 ▼] (선택)                          │
│                                                         │
│            [취소]              [저장]  (상세시 [삭제])   │
└─────────────────────────────────────────────────────────┘
```

**요소**
- 제목(필수), 설명(선택)
- 담당자(필수): 팀원은 본인, 대표는 직원 선택
- 팀, 시작/종료 일시, 반복, 우선순위, 상태, 알림
- 버튼: 취소, 저장 / 상세 시 삭제 추가

## 11. 모바일 대응 (요약)

- **상단**: 햄버거 메뉴 → 대시보드/캘린더/일과/설정
- **캘린더**: 월간은 그리드 유지(터치 스와이프), 주간/일간은 세로 스크롤
- **일정 추가**: 하단 FAB 또는 풀스크린 폼
- **필터(대표)**: 드로어 또는 상단 드롭다운
