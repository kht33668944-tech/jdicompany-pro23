# Vercel 배포 참고

## 푸시 후에도 예전 버전이 보일 때 (필수)

1. **Vercel** → **jdicompany** → **Deployments**
2. **맨 위 배포** 오른쪽 **⋮(점 3개)** 클릭 → **Redeploy**
3. 재배포 창에서 **"Clear build cache"** (또는 "Override the Build Cache") **반드시 체크**
4. **Redeploy** 버튼 클릭 → 빌드 완료까지 2~3분 대기
5. 브라우저 **강력 새로고침** (Ctrl+Shift+R) 또는 시크릿 창으로 접속

사이드바 하단에 **v0.1.2** 가 보이면 최신 배포가 적용된 것입니다.

- **Settings** → **Git** 에서 Production Branch가 `main` 인지 확인
